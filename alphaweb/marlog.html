Centralized Log Management

MarLog (Centralized Log Management)

MarLog เป็นระบบสำหรับจัดเก็บและบริหารจัดการ Log แบบรวมศูนย์ (Centralized Log Management) ที่ออกแบบมาเพื่อช่วยรวบรวมข้อมูล Log จากอุปกรณ์ ระบบเครือข่าย และแอปพลิเคชันต่าง ๆ ภายในองค์กร นำมารวมไว้ในที่เดียวกัน เพื่อให้ง่ายต่อการวิเคราะห์ ประเมิน ตรวจสอบ และสืบค้นย้อนหลังเมื่อเกิดเหตุภัยคุกคามทางไซเบอร์

แหล่งข้อมูล Log ที่รองรับการเชื่อมต่อ

  • Firewall & Network Devices รวบรวม Log จากอุปกรณ์รักษาความปลอดภัยเครือข่าย Firewall, Router, Switch, Load Balancer และ Access Point ทุกแบรนด์
  • Servers & Directory Services จัดเก็บ Log จากระบบปฏิบัติการ Windows Server, Linux Server, Active Directory (AD) และอุปกรณ์จัดเก็บข้อมูลแบบเครือข่าย NAS
  • Endpoints & Antivirus / EDR รับข้อมูลการตรวจจับและวิเคราะห์พฤติกรรมจากซอฟต์แวร์ป้องกันไวรัส และ Endpoint Detection & Response (EDR) บนเครื่องลูกข่าย
  • Applications & Databases รวบรวมประวัติการเข้าใช้งานและการทำงานของแอปพลิเคชัน ตลอดจนฐานข้อมูลต่าง ๆ ในองค์กร (Database Audit Logs)
  • Cloud Services & Syslog รองรับมาตรฐาน Syslog (UDP/TCP/TLS), Windows Event Logs, Web Server Logs และ Cloud Services (AWS / Azure)

Core Features

ฟีเจอร์เด่นของระบบ MarLog

SIEM & Security Operations (SOC)

หน้าแดชบอร์ดสรุปผลการวิเคราะห์ภัยคุกคามแบบเรียลไทม์ ค้นหาความสัมพันธ์ของ Log จากหลากหลายแหล่ง พร้อมเทคโนโลยีวิเคราะห์พฤติกรรมผู้ใช้งานและอุปกรณ์ (UEBA) เพื่อตรวจจับสิ่งผิดปกติได้อย่างแม่นยำ

Asset Intelligence & Discovery

ค้นหาและระบุตัวตนของอุปกรณ์ในระบบเครือข่าย (ทั้ง IT และ OT/IoT) โดยอัตโนมัติ พร้อมการจัดทำฐานข้อมูลรายละเอียดอุปกรณ์ (CMDB) ระบบปฏิบัติการ และประวัติความเสี่ยง

Network Visibility & Topology

สร้างและจำลองแผนผังโครงสร้างเครือข่าย Layer 2 / Layer 3 แบบอัตโนมัติ เพื่อช่วยให้มองเห็นความเชื่อมโยงของอุปกรณ์ ตรวจสอบสถานะการเชื่อมต่อ และปริมาณแบนด์วิดท์แบบเรียลไทม์

Security & Vulnerability Management

วิเคราะห์ช่องโหว่ของอุปกรณ์ต่าง ๆ ในเครือข่ายโดยเทียบกับฐานข้อมูล CVE สากล พร้อมจัดเก็บคะแนนความเสี่ยง (Unified Risk Scoring) และแจ้งเตือนเมื่อพบอุปกรณ์แปลกปลอมในเครือข่าย

Reporting & Compliance Analytics

ระบบสร้างรายงานที่สามารถกำหนดข้อมูลเองได้ (Custom Report Builder) รองรับการส่งออก PDF, CSV, JSON รวมถึงตั้งเวลาส่งรายงานทางอีเมล และสร้างรายงานรองรับมาตรฐาน PDPA, ISO 27001, NIST CSF, PCI-DSS

Automation & SOAR Orchestration

ระบบตอบสนองภัยคุกคามอัตโนมัติ (SOAR) ด้วย Visual Playbook Builder ช่วยให้ทีมผู้ดูแลระบบสามารถกำหนดการตอบสนอง บล็อกไอพี หรือกักกันเครื่องที่ติดมัลแวร์ได้ทันทีผ่าน API

Customer Benefits

ประโยชน์ของการใช้งาน MarLog ในองค์กร

  • รวมศูนย์ข้อมูล Log: ค้นหา วิเคราะห์ และตรวจสอบข้อมูล Log จากทุกอุปกรณ์ได้ในที่เดียว ไม่ต้องแยกเข้าแต่ละอุปกรณ์
  • เพิ่มความรวดเร็วในการตรวจจับ: ตรวจพบความเสี่ยงและเหตุการณ์ผิดปกติได้รวดเร็ว ช่วยป้องกันการโจมตีทางไซเบอร์ที่อาจเกิดขึ้น
  • ตรวจสอบย้อนหลังได้ง่าย (Audit Trail): รองรับการทำดิจิทัลฟอเรนสิกส์ ค้นหาที่มาของเหตุการณ์เพื่อใช้เป็นหลักฐานและหาสาเหตุ
  • สอดคล้องข้อกฎหมายและมาตรฐานสากล: รองรับการปฏิบัติตาม พ.ร.บ. คอมพิวเตอร์ฯ และ พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล (PDPA) รวมถึงมาตรฐาน ISO 27001
  • ลดภาระงานของทีมไอที: ลดเวลาในการรวบรวมและวิเคราะห์ Log ด้วยการใช้ระบบอัตโนมัติแจ้งเตือนตามเงื่อนไขความเสี่ยง
  • สถาปัตยกรรมแบบ Microservices: ออกแบบให้ระบบทำงานได้เสถียร รองรับเครือข่ายขนาดใหญ่ และสามารถทำงานบนระบบ Air-gapped ที่ตัดขาดจากเน็ตได้
